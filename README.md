# Perceptron Demo – Placement Prediction

## 📌 Overview

A simple demo of the **Perceptron algorithm** to classify students as placed (1) or not placed (0) using **CGPA** and **Resume Score**.

## 🚀 Features

* Binary classification using sklearn Perceptron
* Data visualization (Seaborn)
* Decision boundary plotting (mlxtend)

## ▶️ Usage

```bash
pip install numpy pandas matplotlib seaborn scikit-learn mlxtend
jupyter notebook
```

## ⚙️ Steps

1. Load data
2. Visualize data
3. Train Perceptron
4. Plot decision boundary

## 🎯 Output

* Linear separation of classes
* Clear decision regions
